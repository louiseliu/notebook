awesome
=======

awesome 键盘绑定


M-o
将当前程序丢到下一个Screen

M-j
在当前 workspace 切换到下一个程序

M-k
在当前 workspace 切换到上一个程序

M-h
增加当前 workspace 中 主窗口的面积

M-l
减小当前 workspace 中 主窗口的面积

M-r
调用 lua 运行程序

M-m
将当前窗口最大化

M-return
启动 terminal

C-M-left切换到上一个 workspace

C-M-right
切换到下一个 workspace
